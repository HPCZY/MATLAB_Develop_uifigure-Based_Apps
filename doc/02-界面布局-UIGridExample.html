一、创建网格布局

控件较多时，逐一设置位置大小非常繁琐。当需要修改某个控件大小时，也可能影响控件。网格布局工具可以快速实现面板与控件的布局，并且支持等比缩放。

创建网络布局

在UI界面中创建网格布局，默认是2*2的网格
fig = uifigure('Name','网格布局学习','Position',[200,150,400,300]);
gdlt = uigridlayout(fig);
为了更好的观看效果，我们放入四个面板
uipanel(gdlt,'Title','面板一')
uipanel(gdlt,'Title','面板二')
uipanel(gdlt,'Title','面板三')
uipanel(gdlt,'Title','面板四')

设置底色

gdlt.BackgroundColor = [0,1,1];

设置间隔

设置窗口边缘的间隔，分别为 [左，下，右，上]
gdlt.Padding = [20,10,20,25];
设置网格之间的空隙
gdlt.RowSpacing = 5;
gdlt.ColumnSpacing = 10;

二、设置网格分割

快速将界面分为多行多列，并设置每一个格的大小。

按像素分割

直接设置每个网格的具体尺寸，使其不受窗口缩放而改变。
% 生成网格
fig1 = uifigure('Name','网格布局学习1','Position',[200,150,400,300]);
gdlt1 = uigridlayout(fig1);
gdlt1.RowHeight = {200};
gdlt1.ColumnWidth = {200,100};
% 放置控件
uipanel(gdlt1,'Title','面板一')
uipanel(gdlt1,'Title','面板二')

按比例分割

设置每个网格的占比，能够随窗口缩放而改变。格式为“数字+x”。
% 生成网格
fig2 = uifigure('Name','网格布局学习2','Position',[500,150,400,300]);
gdlt2 = uigridlayout(fig2);
gdlt2.RowHeight = {'1x','3x'};
gdlt2.ColumnWidth = {'1x','1x'};
% 放置
uipanel(gdlt2,'Title','面板一')
uipanel(gdlt2,'Title','面板二')
uipanel(gdlt2,'Title','面板三')
uipanel(gdlt2,'Title','面板四')

自适应分割

使用fit，根据控件的尺寸自动设置网格尺寸，完成后不受窗口缩放而改变。
% 生成网格
fig3 = uifigure('Name','网格布局学习3','Position',[800,150,400,300]);
gdlt3 = uigridlayout(fig3);
gdlt3.RowHeight = {'1x','3x'};
gdlt3.ColumnWidth = {'1x','fit'};
% 放置
uipanel(gdlt3,'Title','面板一')
uipanel(gdlt3,'Title','面板二')
uipanel(gdlt3,'Title','面板三')
uipanel(gdlt3,'Title','面板四')

三、放置控件

自动放置

不指定参数的情况下，会按“从左到右，从上到下”的顺序依次放置。
% 生成网格
fig = uifigure('Name','网格布局学习','Position',[200,150,400,300]);
gdlt4 = uigridlayout(fig);
gdlt4.RowHeight = {'1x','1x'};
gdlt4.ColumnWidth = {'1x','1x'};
 
% 放置
uipanel(gdlt4,'Title','面板一')
uipanel(gdlt4,'Title','面板二')
uipanel(gdlt4,'Title','面板三')

指定放置

生成一个网格
fig = uifigure('Name','网格布局学习','Position',[200,150,400,300]);
gdlt = uigridlayout(fig,'RowHeight',{'1x','1x'},'ColumnWidth',{'1x','1x','1x'});
指定每一个控件的位置
pn1 = uipanel(gdlt,'Title','面板一');
pn1.Layout.Row = 1;
pn1.Layout.Column = 1;
pn2 = uipanel(gdlt,'Title','面板二');
pn2.Layout.Row = 1;
pn2.Layout.Column = 2;
还可以指定一个控件占有多个网格（实现更高的自由度）
pn3 = uipanel(gdlt,'Title','面板三');
pn3.Layout.Row = 2;
pn3.Layout.Column = [1,2];
pn3 = uipanel(gdlt,'Title','面板四');
pn3.Layout.Row = [1,2];
pn3.Layout.Column = 3;
