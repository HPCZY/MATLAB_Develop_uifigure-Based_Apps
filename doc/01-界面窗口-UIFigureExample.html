一、创建界面

创建UI界面

fig = uifigure;

设置窗口名

fig.Name = '第一个UI界面';

设置图标

默认为MATLAB官方图标，可以修改为自己喜欢的图片，有两种实现方法：1）直接提供图片的路径；2）先读入图片，再提供矩阵。
推荐使用第一种，能保留图像原有的透明度信息。
fig.Icon = 'icon.png';
第二种方法如下，会损失透明度信息。
icon = imread("icon.png");
fig.Icon = icon;

设置位置大小

分别为 [底部距离，左侧距离，宽，高]
fig.Position = [200,150,400,300];

设置底色

颜色采用RGB表示，如：黑色为[0,0,0]，白色为[1,1,1]，红色为[1,0,0]
fig.Color = [0.2,0.4,0.9];

二、三种初始化方法

以下三种初始方法等价

参数赋值法

fig = uifigure('Name','UI界面','Position',[200,150,400,300],'Color',[0.2,0.4,0.9]);

直接赋值法

fig = uifigure;
fig.Name = 'UI界面';
fig.Position = [200,150,400,300];
fig.Color = [0.2,0.4,0.9];

set赋值法

fig = uifigure;
set(fig,'Name','UI界面');
set(fig,'Position',[200,150,400,300]);
set(fig,'Color',[0.2,0.4,0.9]);

三、缩放控制

创建一个可以自由缩放的界面
fig1 = uifigure('Name','自由缩放','Position',[600,250,300,200]);
创建一个禁止自由缩放的界面
fig2 = uifigure('Name','禁止缩放','Position',[200,250,300,200]);
fig2.Resize = 'off';

四、多窗口访问性

窗口访问有三种类型：normal，modal，alwasontop。

normal 模式

普通窗口，不影响任何窗口。
fig3 = uifigure('Name','normal窗口1','WindowStyle','normal');
fig4 = uifigure('Name','normal窗口2','WindowStyle','normal');

alwaysontop 模式

普通窗口，界面会处于其他所有类型窗口之上（两个alwaysontop窗口之间互不影响）。
fig5 = uifigure('Name','alwaysontop窗口1','WindowStyle','alwaysontop');
fig6 = uifigure('Name','alwaysontop窗口2','WindowStyle','alwaysontop');

modal 模式

高级窗口，会禁用在此之前生成的所有窗口（对之后生成的窗口没有影响），关闭后可解禁前置窗口。
fig7 = uifigure('Name','modal窗口1','WindowStyle','modal');
fig8 = uifigure('Name','modal窗口2','WindowStyle','modal');
