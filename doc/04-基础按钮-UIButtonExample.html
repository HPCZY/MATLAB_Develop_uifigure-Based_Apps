一、创建按钮

创建按钮

% 创建UI界面
fig = uifigure;
 
% 创建按钮
bt = uibutton(fig);
 
% 设置位置
bt.Position = [100 100 50 50];
 
% 设置文本
bt.Text = '按钮';
 
% 设置图标
bt.Icon = '';
 
% 设置底色
bt.BackgroundColor = [0.9,0.9,0.8];

设置类型

该控件有两个类型：普通按钮（默认）、状态按钮。
% 创建普通按钮
bt = uibutton(fig);
% 创建状态按钮
bt = uibutton(fig,'state');
普通按钮每次点击给于反馈，状态按钮会根据点击在两种状态之间切换。

二、对齐规则

字体默认居中，可以通过参数调节， 'center' (default) | 'left' | 'right' | 'top' | 'bottom'
图标默认靠左，可以通过参数调节，'left' (default) | 'right' | 'center' | 'leftmargin' | 'rightmargin' | 'top' | 'bottom'
创建一个图像框组
fig = uifigure('Name','按钮学习','Position',[200,150,300,200]);
bt1 = uibutton(fig,'Position',[0,100,100,100],'Text','按钮一','Icon','icon.png');
bt2 = uibutton(fig,'Position',[100,100,100,100],'Text','按钮二','Icon','icon.png');
bt3 = uibutton(fig,'Position',[0,0,100,100],'Text','按钮三','Icon','icon.png');
bt4 = uibutton(fig,'Position',[100,0,100,100],'Text','按钮四','Icon','icon.png');
设置对齐方式
% bt1
% 默认
% bt2
bt2.IconAlignment = 'center';
% bt3
bt3.HorizontalAlignment = 'left';
bt3.IconAlignment = 'rightmargin';
% bt4
bt4.HorizontalAlignment = 'right';
bt4.VerticalAlignment = 'top';
bt4.IconAlignment = 'bottom';
注意：图标的位置是字体为参照，而不是以按钮框为参照。

三、交互

创建界面
fig = uifigure('Name','按钮学习','Position',[200,150,200,200]);
% 普通按钮
bt1 = uibutton(fig,'Text','普通按钮','Position',[50,100,100,50]);
% 状态按钮
bt2 = uibutton(fig,'state','Text','状态按钮','Position',[50,50,100,50]);

普通按钮

设置点击回调，点击触发func1
bt1.ButtonPushedFcn = @func1;

状态按钮

设置点击回调，状态改变触发func2
bt2.ValueChangedFcn = @func2;
创建回调
% 回调函数包含： 1）src—按钮本体；2）event—按钮触发的事件
function func1(src,event)
disp('普通按钮')
end
 
function func2(src,event)
 
if src.Value % 即 bt2.Value
disp('状态按钮按下')
else
disp('状态按钮弹起')
end
 
end

fig = uifigure('Name','按钮学习','Position',[200,150,200,200]);
% 普通按钮
bt1 = uibutton(fig,'Text','普通按钮','Position',[50,100,100,50]);
% 状态按钮
bt2 = uibutton(fig,'state','Text','状态按钮','Position',[50,50,100,50]);

bt2.ValueChangedFcn = @func2;
 
function func1(src,event)
disp('普通按钮')
end
 
function func2(src,event)
 
if src.Value % 即 bt2.Value
disp('状态按钮按下')
else
disp('状态按钮弹起')
end
 
end
