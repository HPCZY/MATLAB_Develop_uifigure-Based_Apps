一、创建标签

用于给界面或（不含文字的）组件添加文字说明。

创建标签

% 创建UI界面
fig = uifigure('Name','按钮组学习','Position',[500,150,300,200]);
 
% 创建标签
lb = uilabel(fig);
 
% 文字内容
lb.Text = '标题';
 
% 设置位置
lb.Position = [50,50,100,100];
 
% 设置背景颜色
lb.BackgroundColor = [1,1,1];
uifigure中大多数组件都含有BackgroundColor属性，仅在这里讲一次（特殊情况会说明）

二、字体设置

uifigure中大多数含文字的组件都拥有以下属性设置，仅在这里讲一次（特殊情况会说明）。
其中，
  • 字体：对应windows系统上存在的；
  • 粗细： 'normal' （默认） | 'bold'；
  • 角度： 'normal' （默认） | 'italic'。
% 字体名称
lb.FontName = '宋体';
 
% 字体大小
lb.FontSize = 24;
 
% 字体粗细
lb.FontWeight = 'bold';
 
% 字体角度
lb.FontAngle = 'italic';
 
% 字体颜色
lb.FontColor = [0 0 0.3];

三、文本设置

uifigure中大多数含文字的组件都拥有以下属性设置（除特殊说明外）。仅在这里讲一次。

文本对齐方式

  • 水平对齐方式：'left' （默认） | 'right' | 'center'
  • 垂直对齐方式：'center' （默认） | 'top' | 'bottom'
  • 自动换行：'off' （默认） | on/off 逻辑值
% 文字对齐
lb.HorizontalAlignment = 'center';
lb.VerticalAlignment = 'top';
 
% 开启自动换行
lb.WordWrap = 'on';
也可利用元胞数组实现手动换行，如下代码
fig = uifigure('Name','按钮组学习','Position',[500,150,350,200]);
lb = uilabel(fig,'Position',[0,0,350,200],'FontSize',16,'VerticalAlignment','top');
lb.Text ={'振荡电路是电子技术的一个重要组成部分，';
'正弦波振荡器广泛应用于广播、电视、通讯，';
'工业自动控制，测量表计，';'以及高频加热，探伤等等方面。'};

文本解释器

按照特殊的格式解析文本内容，使其具有更高的自由度。
  • 解释器：'none' （默认） | 'tex' | 'latex' | 'html'
% 创建UI界面
fig = uifigure('Name','按钮组学习','Position',[500,150,300,200]);
 
% 创建标签
lb1 = uilabel(fig,'Position',[25,150,250,40],'FontSize',24,'FontWeight','bold');
lb2 = uilabel(fig,'Position',[25,100,250,40],'FontSize',16);
lb3 = uilabel(fig,'Position',[25,20,250,80],'FontSize',16);
 
% 设置文本
lb1.Text = '正弦波';
lb1.HorizontalAlignment = 'center';
 
lb2.Text = '$$Y = Asin(\omega t+\phi)$$';
lb2.HorizontalAlignment = 'center';
lb2.Interpreter = 'latex';
 
lb3.Text = "<font style='color:red;'>正弦波</font>是频率成分最为单一的一种<br>信号，波形是数学上的正弦曲线。";
lb3.Interpreter = 'html';
lb.VerticalAlignment = 'top';
lb3.WordWrap = 'on';
仅标签含有此功能，类似的文本框和编辑框都没有。
