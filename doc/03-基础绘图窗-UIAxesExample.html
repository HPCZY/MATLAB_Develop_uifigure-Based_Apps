一、创建绘图坐标轴

创建坐标轴

fig = uifigure('Name','绘图窗学习','Position',[200,150,400,300]);
ax = uiaxes(fig);

设置位置

ax.Position = [50,50,300,200];

设置颜色

ax.Color = [0.5,0.5,1];

二、绘图

先创建一个坐标轴组
fig = uifigure('Name','绘图窗学习1','Position',[200,150,800,300]);
ax1 = uiaxes(fig,'Units','normalized','Position',[0,0,0.33,1]);
ax2 = uiaxes(fig,'Units','normalized','Position',[0.33,0,0.33,1]);
ax3 = uiaxes(fig,'Units','normalized','Position',[0.67,0,0.33,1]);

显示图像

可以利用 imshow | imagesc | image 等常见函数在坐标轴中显示图像
img = imread('cameraman.tif');
imshow(img,'Parent',ax1);

二维作图

可以利用 plot | scatter | bar 等常见函数在坐标轴中绘制数据
x = 0:0.01:2*pi;
y = sin(2*x);
plot(x,y,'Parent',ax2)

三维作图

可以利用 plot3 | scatter3 | surf 等常见函数在坐标轴中绘制数据
[X,Y] = meshgrid(-5:.5:5);
Z = Y.*sin(X) - X.*cos(Y);
surf(X,Y,Z,'Parent',ax3);
