一、创建编辑框

在文本框的基础上，对输入内容设置限制条件；主要用于输入。

创建编辑框

% 创建UI界面
fig = uifigure('Name','编辑框学习','Position',[500,150,300,200]);
 
% 创建编辑框
ef = uieditfield(fig);
 
% 设置位置
ef.Position = [25,25,150,150];
 
% 设置背景颜色
ef.BackgroundColor = [1,1,1];

文本设置

当输入为空时，可以显示默认文本（一旦输入则会清楚默认文本）；
% 设置默认文本
ef.Placeholder = '请输入文字';
与uilabel不同，本文框规定从顶端开始显示文字，所以只有水平对齐
  • 'left' （默认） | 'right' | 'center'
设置居中
% 文字对齐
ef.HorizontalAlignment = 'center';

文本状态

文本框有三种状态，通过两个参数设置
  • 可编辑（默认）
  • 只读
  • 禁用
% 禁止编辑
ef.Editable = 'off';
 
% 禁用
ef.Enable = 'off';

二、输入控制

编辑框有两种控制
  • 字符数：[0 Inf] （默认） | 二元素数值数组
  • 文本内容：'text' （默认） | 'letters' | 'digits' | 'alphanumerics'
% 创建界面
fig = uifigure('Name','编辑框学习','Position',[500,150,300,200]);
 
% 创建组件
uilabel(fig,'Text','创建账号：','Position',[20,140,160,40]);
uilabel(fig,'Text','设置密码：','Position',[20,60,160,40]);
cb1 = uieditfield(fig,'Value','','Position',[20,100,160,40],'Placeholder','请输入任意字符');
cb2 = uieditfield(fig,'Value','','Position',[20,20,160,40],'Placeholder','8位以上数字');
 
% 设置规则
cb2.CharacterLimits = [8,Inf];
cb2.InputType = 'digits';

三、交互

编辑框有两种回调
  • 编辑完成回调：修改文本内容，并确认而产生的回调；
  • 正在编辑回调：正在修改文本内容，产生回调。
创建界面
% 创建界面
fig = uifigure('Name','文本框学习','Position',[500,150,300,200]);
 
% 创建组件
lb = uilabel(fig,'Text','请侧输入：','Position',[20,160,160,40]);
cb = uieditfield(fig,'Value','','Position',[20,25,160,120],'BackgroundColor',[0.8,0.8,0.8]);
% 设置回调
cb.ValueChangedFcn = @func1;
cb.ValueChangingFcn = @func2;
创建回调
% 回调函数包含： 1）src—按钮本体；2）event—按钮触发的事件
function func1(src,event)
% 编辑完变灰
src.BackgroundColor = [0.8,0.8,0.8];
end
 
function func2(src,event)
% 编辑时变白
src.BackgroundColor = [1,1,1];
end

三、更多

编辑框与文本框的区别：1）编辑框仅一行；2）文本框可以随意输入内容，支持多行。

完整代码

% 创建界面
fig = uifigure('Name','文本框学习','Position',[500,150,300,200]);
 
% 创建组件
lb = uilabel(fig,'Text','请侧输入：','Position',[20,160,160,40]);
cb = uieditfield(fig,'Value','','Position',[20,25,160,120],'BackgroundColor',[0.8,0.8,0.8]);
% 设置回调
cb.ValueChangedFcn = @func1;
cb.ValueChangingFcn = @func2;
 
% 回调函数包含： 1）src—按钮本体；2）event—按钮触发的事件
function func1(src,event)
% 编辑完变灰
src.BackgroundColor = [0.8,0.8,0.8];
end
 
function func2(src,event)
% 编辑时变白
src.BackgroundColor = [1,1,1];
end
